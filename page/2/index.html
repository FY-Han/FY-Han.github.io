AI 第一节

基础操作

alt+滚轮 放缩    空格+左键 平移     Ctrl+滚轮 水平移动      TAB 全屏       Ctrl+0 恢复
空格+鼠标拖拽 抓手      Ctrl+r 标尺     Ctrl+；参考线       Ctrl+" 网格    Ctrl
Ctrl+Y 轮廓模式     Ctrl+Alt+Y 像素模式     Ctrl+Shift+AltY 叠印模式

Ctrl+2 锁定图层，对应ctrl+alt+2

  1. 选择工具
    Ctrl 临时选择工具
    V 选择工具 shift 等比拉伸，alt 以中心点拉伸
    A 直接选择工具，可以移动变形（需双击） 也可以使用套索工具
    使用 ctrl + shift +b锁住定界框（失去白色小点，不能改变大小了），ctrl+h隐藏路径
    色彩辨别（眯眼），用魔棒
    Ctrl+C Ctrl+B 原位粘贴

  2. 填色描边
    前后互换 X 颜色互换 Shift+X 比例缩放工具：等比圆滑放缩 倾斜工具：点住不变的点，进行倾斜
    Ctrl+d 重复操作 Alt 复制 Alt+Shift
    颜色渐变

  1. 钢笔工具
    先点起点，然后在落点拉切线
    长按左键拉出切线，用ctrl暂时取消，双击空白 按住alt键拉切线 按shift辅助直线与45°倍数角
    按shift+c完成并选择锚点工具
    钢笔工具练习网站：https://bezier.method.ac/

  2. 图层
    Ctrl+[ 上下移动图层中子图层位置关系

  3. 文字
    Ctrl+T 文本编辑框 Alt+上下左右调整字间距行间距 Shift+T 编辑文本 可以在创建的路径上编辑文字
    Ctrl+Shift+O 将文字转成文本轮廓（才能进行拖拽变形）
    路径查找器合并多个路径
    如果有奇怪的白色在比如里字的每个“口”里（不知为何），可以栅格化解决问题

  4. 效果
    旋转，比如六边形的内旋，然后
    混合工具：形成图形的渐变,也可以形成立体效果（先后点击两个物体即可）

  5. 重新着色图稿
    选中后，编辑——颜色——重新着色图稿

    • 注意填色是填充路径的颜色（不太懂原理），如果给裁剪后的图形，如圆，进行局部填色会导致填路径内。进行扩展外观后，失去边缘路径，重新把所有性状变成“内容”

      • 扩展（不要勾选填充，否则填充和线被拆分开），最后使用路径查找器拼合

    • 轮廓化描边

      • 在路径中，将描边变成可以选中的路径，进而可以渐变颜色，或者加图案

gmx入门3

继续探索

网上搜索，注意到提示需要full trajectory file
结果突然发现80min后出结果了，

trr分析与结果生成

  1. 使用nmeig

    频率生成基于quasi harmonic approximation

  2. 进一步可以研究anaeig

标准流程

SPDBV前处理

  1. pdb转gro文件，生成拓扑以及水盒子

    gmx pdb2gmx -f fws.pdb -o fws.gro -p fws.top -i fws.itp -ff amber99sb-ildn -water tip3p -ignh

  2. 生成周期性盒子
    gmx editconf -f fws.gro -o fws-PB.gro -bt dodecahedron -d 1.2
  3. 真空能量最小化（三步走，先steep EMTOL50 再cg到10 再l-bfgs到10e-6）
    • 3W步 steep法到EMTOL 20
      gmx grompp -f fws-VM1.mdp -c fws-PB.gro -p fws.top -o fws-VM1.tpr -maxwarn 1
      gmx mdrun -v -deffnm fws-VM1
    • 8w步 cg法到EMTOL 10(可以不用也许)
      gmx_d grompp -f fws-VM2.mdp -c fws-VM1.gro -p fws.top -o fws-VM2.tpr -maxwarn 1
      gmx_d mdrun -v -deffnm fws-VM2
    • 10w步 l-bfgs法
      gmx_d grompp -f fws-VM3.mdp -c fws-VM2.gro -p fws.top -o fws-VM3.tpr -maxwarn 1
      gmx_d mdrun -v -deffnm fws-VM3
  4. 跑长时md？
    • 后台运行
      gmx_d grompp -f npt-nopr-md.mdp -c fws-VM3.gro -p fws.top -o npt-nopr.tpr
  5. 跑Hessian
    • gmx_d grompp -f Hessian.mdp -c fws-VM3.gro -p fws.top -o Hessian.tpr -maxwarn 1
      gmx_d mdrun -v -s Hessian.tpr -o H-traj.trr -c confout.gro -e ener.edr -g md.log -mtx nm.mtx
