gromacs学习——mdp文件编写

mdp文件注解小结

来源：踏雪无痕
简要摘录，详细的去看文件

; 预处理

title= # 最长64字符
cpp # 预处理器（默认cpp）
include # 拓扑文件引用其他文件的路径
define= # -DFLEXIBLE（用柔性水模型，共轭梯度必要；或-DPOSRES（用于位置限制md）

; 模拟控制

integrator #
md：leap-frog做md
sd：stochastic dynamics做md
bd：Euler方法啥的做md
steep：最速下降能量优化 同时设置能量优化最大位移参数 emstep 由emtol决定最大容忍度
cg：共轭梯度能量优化，与最速联用，对能量优化常态分析时用双精度
l-bfgs：不并行，类似cg，收敛快
nm：对tpr文件常态分析，必须为双精度GROMACS
tpi：插入测试粒子

; 开始时间与步长 单位ps
tinit=0 # 模拟开始时间，默认0，min不用，只用于md sd bd
dt=0.002 # 步长2fs，默认1fs，同上
nsteps=50000# 默认0

; run continuation or redoing part of a run
init_step # 第i步时间可以算

; center of mass remove
comm_mode = Linear # 质心移动 Linear 平动，Angular转动平动，No不操作

; numer of steps for mass center removal
nstcomm = 10 # 操作频率，默认0，单位step

Langevin动力学

; friction coefficient and random seed
bd_fric = 0
ld_seed = 1993

Gaussian 入门

Gaussian入门流程

  1. 装软件可以用su -命令获取管理员身份
    path里/home表示根目录，最开始要有/
    绝对路径 ~/.bashrc打开
    .bashrc修改完后 source ~/.bashrc 而不必重新进终端，或者断开连接 但记得要重启一下

    没装Gaussian 只装了view submit是灰的，可以点击edit导出保存
    有机分子建构

  2. 水分子频率流程
    • 用win里的view建立gjf输入文件
      • view里画好在紫框左键点一下
      • 右键calculate 然后setup
        • Jobtype：opt 能量优化 可以再加一个freq
        • 方法基态 hartree-fock
        • 基组3-21G
      • 弹出窗口改linux路径
    • 导入linux
      • 之前不改路径就报错
        ntrex1: Bad file descriptor
        不是大小写问题，打开文件看看，可能是路径问题，保留了win路径
        也有可能因为是win生成的文本文件，需要把unix格式转为dos格式，使用dos2unix命令
        dos2unix file1 file2
    • 生成二进制chk和log文件
      用命令生成fchk文本文件，chk是二进制文件
        formchk xxx.chk
    • 可以把log导入view然后在result选项选frequency
